# Rock Identification System Workflow

## Home Dashboard

- **Entry point for all users**
  - Branches to major features:
    - Identify Rocks
    - Learn
    - Community
    - Rock Library
    - Profile & Settings

---

## Identify Rocks Module

### Methods

- **View Map**

  - Select location
  - View table of rocks found in area
  - Select rock to display properties

- **Visual Matcher**

  - Upload photo or image
  - Auto-detect rock properties

- **Project Rock**
  - Assign rock to lab log
  - Teamwork evaluation

---

## Rock Library

### Navigation

- Search/filter rocks by:
  - Location
  - Mineral composition
  - Rock type
- Display geological maps by region
  - Rocks are colored per region
  - Map view for selected material
- **Rock Info Card**
  - Display details for selected rock
  - Pin/save individual rock cards

---

## Learning Section

### Features

- Lessons: rock types (igneous, sedimentary, metamorphic)
- Field qualification tips
- Mini quizzes (difficulty selection)
- Geology paths and guides

## Community Module

### Post Feed Board

- Recent uploads and posts
- Options for viewing, commenting, and sharing
- Upload photos with context and tags
- Leaderboards for user contributions

## Profile & Settings

### Profile Management

- Account details and preferences
- Password management
- Bookmarked pins/posts
- Downloaded content/message history
- Feedback section
